# Supported languages

Jupytext works with notebooks in any of the following languages:

- Bash
- C#
- C++
- Clojure
- Coconut
- F#
- Groovy
- IDL
- Java
- Javascript
- Julia
- Matlab
- Octave
- PowerShell
- Python
- q/kdb+
- R
- Robot Framework
- Rust/Evxcr
- Scala
- Scheme
- Script of Script
- TypeScript

Extending Jupytext to more languages should be easy, see [CONTRIBUTING.md](https://github.com/mwouts/jupytext/blob/master/CONTRIBUTING.md).
